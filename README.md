**GOAL:** Reduce 90% drudgery and Time consumption.

## Assumptions

-  Assumes paragraph entities have a "Preview" display mode.
-  Assumes media entities have a "Inline Entity Form" form mode.
-  Assumes the following entity browsers
  -  image_media_browser
  -  video_media_browser
